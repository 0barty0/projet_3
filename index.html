Véloloc Lyon

Bienvenue chez Véloloc Lyon

Notre service de location de vélo est disponible dans 340 stations réparties sur toute la métropole de Lyon.

Ce diaporama vous explique comment effectuer une réservation.

Bonne route

La carte

Utilisez la carte pour trouver la station de votre choix. Celle-ci vous indique le nombre de stations dans chaque zone.

Zoomez sur la carte pour affiner votre recherche puis cliquez sur la station de votre choix.

La fiche d'informations

Les détails de la station s'affiche sur le côté.

Vous pouvez vérifier la disponibilité d'un vélo à cette station avant d'effectuer la réservation en cliquant sur le bouton.

La signature

Pour confirmer votre réservation il vous faut signer dans le cadre prévu à cet effet avant de cliquer sur le bouton "confirmer".

La réservation

Votre réservation de vélo s'affiche sous la carte. Celle-ci est valable pour 20 minutes.

Vous pouvez l'annuler à tout moment ou effectuer une autre réservation pour la remplacer.
